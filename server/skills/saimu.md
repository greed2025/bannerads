# 債務整理 スキルファイル

## NG表現リスト

| NG表現 | 理由 | 代替表現 |
|--------|------|---------|
| 必ず解決 | 断定的すぎる（景品表示法抵触の恐れ） | 解決に向けてサポート |
| 借金ゼロ | 誤解を招く | 負担軽減の可能性 |
| 絶対に | 断定的すぎる | 多くの方が |
| 100%減額 | 虚偽表現 | 減額の可能性 |
| 今すぐ借金消滅 | 誇大表現 | 今すぐご相談を |

## 必須キーワード
- 無料相談
- 秘密厳守
- 取り立てストップ
- 任意整理
- 過払い金

## 推奨する構成パターン
1. **共感導入**: 借金の辛さに寄り添う
2. **問題提起**: このまま放置するリスク
3. **解決策提示**: 債務整理という選択肢
4. **実績・信頼**: 相談件数、解決事例
5. **CTA**: 無料相談への誘導

## トーン補足
- 重すぎず、希望を持たせる
- 「恥ずかしい」「情けない」などネガティブ感情を煽らない
- プロフェッショナルかつ親身な印象
