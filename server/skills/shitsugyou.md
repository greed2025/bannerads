# 失業保険 スキルファイル

## NG表現リスト

| NG表現 | 理由 | 代替表現 |
|--------|------|---------|
| 必ずもらえる | 断定的すぎる | 受給の可能性があります |
| 簡単に | 誤解を招く | スムーズに |
| 裏技 | 不正を連想させる | 知られていない制度 |
| 会社を辞めても安心 | 無責任 | 退職後の生活をサポート |

## 必須キーワード
- 失業手当
- 受給資格
- ハローワーク
- 離職票
- 給付日数

## 推奨する構成パターン
1. **共感導入**: 退職後の不安に寄り添う
2. **情報提供**: 失業保険の基本説明
3. **メリット提示**: 受給することで得られる安心
4. **具体的手順**: 申請の流れ
5. **CTA**: 詳細確認への誘導

## トーン補足
- 不安を煽りすぎない
- 正確な情報提供を心がける
- 前向きな転職をサポートする姿勢
