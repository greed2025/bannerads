# クリニック スキルファイル

## NG表現リスト

| NG表現 | 理由 | 代替表現 |
|--------|------|---------|
| 必ず治る | 医療広告規制違反 | 改善が期待できます |
| 最高の | 比較優良広告禁止 | 高品質な |
| 日本一 | 根拠なき優位性表現 | 多くの実績 |
| 絶対安全 | 断定的すぎる | 安全性に配慮 |
| 副作用なし | 虚偽表現 | 副作用リスクを最小限に |

## 必須キーワード
- カウンセリング
- 無料相談
- 経験豊富
- プライバシー配慮
- アフターケア

## 推奨する構成パターン
1. **悩み共感**: 美容・健康の悩みに寄り添う
2. **解決策提示**: 施術・治療の説明
3. **安心材料**: 医師の経験、実績
4. **プロセス説明**: 施術の流れ
5. **CTA**: 無料カウンセリング予約

## トーン補足
- 医療広告ガイドラインを遵守
- 過度な期待を持たせない
- 専門性と親しみやすさのバランス
