# ライティングスタイル設定

## トーン
- 親しみやすく、でも専門性を感じさせる
- 語尾は「〜です」「〜ます」を基本に
- 押し付けがましくなく、自然に誘導する

## 好みの表現
- 「あなた」より「お客様」「ご相談者様」を使用
- 「できます」より「可能です」を優先
- 「〜しましょう」より「〜してみませんか」

## 避ける表現
- 過度に硬い表現（法律用語の羅列など）
- 曖昧な言い回し
- 煽りすぎる表現

## フォーマット
- 見出しは短く明確に
- 箇条書きを積極的に活用
- 1文は50文字以内を目安に

## 構成の好み
- 冒頭で共感・問題提起
- 中盤で解決策・メリット提示
- 末尾で具体的なCTA（行動喚起）
